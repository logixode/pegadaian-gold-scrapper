# pegadaian-gold-data

## Prerequisite
```
- Node.js
- vue-cli
- Internet Connection
```

## Installation
1. For Client
```
npm install
```
2. For Backend
```
cd scrapper
npm install
```

### Compiles and hot-reloads for development
1. Client-side
```
npm run serve
```
2. Backend-side
```
node scrapper/main.js
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
